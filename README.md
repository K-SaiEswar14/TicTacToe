# Tic Tac Toe 🎮

A classic Tic Tac Toe game built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies, just the browser.

---

## 📖 About

This is a simple two-player Tic Tac Toe game that runs entirely in the browser. Players take turns marking cells on a 3×3 grid. The first player to align three marks in a row, column, or diagonal wins the game. A restart button lets players reset the board and play again instantly.

---

## 🚀 Demo

Open `index.html` directly in your browser — no server or build step required.

## 🎮 How to Play

1. **Clone or download** this repository.
2. Open `index.html` in any modern web browser.
3. **Player X** goes first — click any empty cell to place your mark.
4. Players alternate turns until one wins or the board is full (draw).
5. The result is displayed below the board.
6. Click **Restart Game** to reset and play again.

---

## ✨ Features

- Two-player gameplay (X and O)
- Win detection across all rows, columns, and diagonals
- Draw detection when the board is full
- Result announcement after each game
- One-click restart to reset the board

## 🖥️ Getting Started

### Prerequisites

Just a modern web browser (Chrome, Firefox, Edge, Safari).

### Installation

```bash
# Clone the repository
git clone https://github.com/K-SaiEswar14/TicTacToe.git

# Navigate into the project folder
cd TicTacToe

# Open in your browser
open index.html   # macOS
start index.html  # Windows
xdg-open index.html  # Linux
```
